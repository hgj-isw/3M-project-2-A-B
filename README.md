# ISW Hoogeland - T&T Projectportaal

Deze repository bevat een statische website (HTML/CSS/JS) die klaar is voor GitHub Pages.

## Inhoud
- `index.html` - hoofdpagina
- `styles.css` - styling
- `script.js` - interactie (mobiel menu, back-to-top)
- `assets/docs/` - lesmateriaal en projectbestanden (nu placeholders)
- `assets/img/favicon.svg` - favicon
- `404.html` - nette fallbackpagina
- `robots.txt` en `sitemap.xml` - basis SEO

## Publiceren op GitHub Pages
1. Maak een nieuwe repository op GitHub.
2. Upload alle bestanden uit deze map.
3. Ga naar **Settings > Pages**.
4. Selecteer:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (of `master`)
   - **Folder:** `/ (root)`
5. Sla op. Je site komt online op `https://<username>.github.io/<repo>/`.

## Bestanden vervangen
Je kunt placeholderbestanden in `assets/docs/` vervangen door echte bestanden.

Tip:
- Gebruik bij voorkeur dezelfde bestandsnamen, dan hoef je geen links aan te passen.
- Voor PDF's kun je direct `.pdf` bestanden uploaden in dezelfde map.
